1990
Web 1.0

Web 1.0 – La web estática y de solo lectura

Año de aparición: ~1990

Primera versión de la web. Páginas estáticas, hipertexto básico, solo lectura. El usuario es un receptor pasivo. El contenido lo controla totalmente el webmaster.

  • Solo consumo de información
  • Centralizada y secuencial
  • Tecnologías: HTML básico, GIF animados, frames
  • Ejemplos: Geocities, Yahoo! Directory, portales informativos

Küster & Hernández (2013); Latorre (2018); Argüelles Toache (2022)

2004
Web 2.0

Web 2.0 – La web social y colaborativa

Término acuñado por: Tim O’Reilly (2004)

El usuario pasa de consumidor a creador (prosumidor). Nace la web participativa.

  • Redes sociales, blogs, wikis, YouTube
  • Interacción bidireccional
  • Tecnologías: AJAX, RSS, APIs
  • Ejemplos: Facebook, Wikipedia, Twitter

O’Reilly (2004); Latorre (2018); Argüelles Toache (2022)

2010
Web 3.0

Web 3.0 – La web semántica e inteligente

La web entiende el significado de los datos y los personaliza.

  • Web semántica + Big Data + IA
  • Personalización profunda
  • Descentralización (blockchain)
  • Ejemplos: Siri, recomendaciones, Ethereum

Ramírez León & Peña Arcila (2011); Argüelles Toache (2022)

2016→
Web 4.0

Web 4.0 – La web simbiótica y omnipresente

Fusión total entre humano y máquina. La web anticipa y actúa por nosotros.

  • IA avanzada (Deep Learning)
  • IoT, 5G/6G, realidad extendida
  • Agentes inteligentes siempre activos
  • Ejemplos: Metaverso, ciudades inteligentes

Latorre (2018); Argüelles Toache (2022)

Evolución de las Plataformas Educativas (LMS)

Web 1.0 (1995–2000)
Nacimiento real del LMS

• WebCT (1995), Mallard, CyberProf
• CourseInfo (pre-Blackboard)
• WOLF

Profesor-céntricas, contenido estático
Web 2.0 (2002–2012)
El LMS moderno y colaborativo

• Moodle (2002)
• Sakai, Blackboard avanzado
• Edmodo, Schoology

Wikis, blogs, e-learning 2.0
Web 3.0 (2015→)
Inteligencia y personalización

• Learning Analytics
• Rutas adaptativas
• Plugins IA
• Credenciales blockchain
Web 4.0 (futuro)
Educación inmersiva

• Clases en metaverso
• VR/AR conectada
• Asistentes IA permanentes
• IoT en aula física
